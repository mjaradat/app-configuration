# Client Runtime Configuration Onboarding Template

Your Cubes Hub client key is:

```text
{clientKey}
```

Open Cubes Hub. The first screen will ask for the Client Runtime Configuration key.

You can:
- enter the key to load your client configuration, branding, URLs, and enabled modules
- skip the key and continue with the default Cubes configuration

The key is not a password. It only loads public client configuration.
